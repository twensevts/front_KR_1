# УберМагаз

Проект интернет магазина.

## Выполненная работа

1. Практика 1
  Сверстана витрина товаров, настроены стили на SCSS, реализована адаптивная сетка карточек.

2. Практика 2
  Реализован backend на Node.js и Express с REST API для работы с товарами.
  Поддерживаются операции создания, чтения, изменения и удаления.

3. Практика 3
  Протестирован API с помощью Postman.

4. Практика 4
  Реализован клиент на React: список товаров, добавление, редактирование и удаление через API.

5. Практика 5
  Подключена Swagger документация и интерактивное тестирование API по адресу /api-docs.

## Функционал сайта

- Просмотр каталога товаров в виде карточек.
- Добавление нового товара через форму.
- Редактирование данных существующего товара.
- Удаление товара из каталога.
- Получение и тестирование API через Swagger UI.

## Технологии

- Node.js
- Express
- React
- Axios
- SCSS
- Swagger (swagger-jsdoc, swagger-ui-express)

## Запуск проекта

Установка зависимостей:

```bash
npm install
npm --prefix client install
```

Запуск:

```bash
npm run dev
```

Адреса после запуска:

- Backend: http://localhost:3000
- Frontend: http://localhost:3001
- Swagger: http://localhost:3000/api-docs
